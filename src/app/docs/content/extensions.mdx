DX Extensions provides a host adapter registry that connects DX tools to 17+ different platforms including editors, design tools, and productivity suites.

## Supported host adapters

| Platform | Type | Features |
|----------|------|----------|
| VS Code | Editor | Commands, themes, language support |
| Cursor | AI Editor | AI-assisted coding, inline suggestions |
| Windsurf | AI Editor | Agentic coding workflows |
| Zed | Editor | Native DX Code integrations |
| Chrome | Browser | DevTools, web extension |
| Edge | Browser | DevTools, web extension |
| Firefox | Browser | Web extension |
| Obsidian | Knowledge | Note-taking, knowledge management |
| Figma | Design | Design-to-code, asset export |
| Blender | 3D | 3D modeling pipelines |
| Unity | Game Dev | Game development workflows |
| Unreal | Game Dev | Game engine integration |
| DaVinci Resolve | Video | Video editing pipelines |
| Adobe | Creative | Creative suite integration |

## Usage

```bash
# List available extensions
dx extensions list

# Install extension for a host
dx extensions install vs-code

# Configure extension
dx extensions configure cursor

# Update extensions
dx extensions update
```
